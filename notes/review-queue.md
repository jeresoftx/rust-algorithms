# Cola de Repaso

## Cómo Usarla

Registrar aquí problemas para repetir después de simulacros o errores detectados.

Estados:

- `pendiente`: aún no se repite.
- `repetido`: ya se resolvió otra vez.
- `dominado`: salió limpio bajo tiempo.
